## How to sign up

1. Pick an empty row (or add a new one)
2. Tell the group in chat
3. Open a PR or ask the repo owner to add your name here

## Before your session

- [ ] Read the [tutor checklist](./README.md#tutor-checklist-when-its-your-turn) in the main README
- [ ] Confirm you have **write access** to the repo

## After your session

- [ ] Update `<Topic>/<topic>.md` (theory)
- [ ] Update `<Topic>/README.md` (session notes + HW + tests)
- [ ] Create the homework issue (use the **Homework** template)
- [ ] Paste the issue link in the table above
- [ ] Post the issue link in the class group chat
